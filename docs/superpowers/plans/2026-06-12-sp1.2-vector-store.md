> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development. Spec: `docs/superpowers/specs/2026-06-12-sp1.2-vector-store-design.md` (locked: interface + 2 backends · `VECTOR_BACKEND` default pgvector · hit-then-hydrate · raw-vector getter · migration tool).
